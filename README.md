# whatsapp_ui

A completely Responsive Whatsapp UI App- works on Android, iOS, Web & Desktop! Works as WhatsApp App on Smaller Screen Sizes and WhatsApp Web on Bigger Screen Sizes!




## Deployment

To deploy this project run

```bash
  flutter run
```
